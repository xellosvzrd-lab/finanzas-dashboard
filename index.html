¿Cómo usar?
Pegá filas copiadas desde Excel o Google Sheets (las columnas se separan con Tab). También podés usar punto y coma como separador.

Orden de columnas:
#ColumnaFormato / ValoresObligatorio
1FechaDD/MM/YYYY o YYYY-MM-DD✅
2TipoGasto o Ingreso✅
3CategoríaCualquier categoría existente✅
4MontoNúmero positivo (ej: 1500.50)✅
5DescripciónTexto libre—
6ResponsabilidadMío / Compartido / De Ama— (default: Mío)
7FuenteGalicia, HSBC, etc.—
8MonedaARS (pesos) o USD (dólares)— (default: ARS)

Ejemplo de fila (ARS):
15/01/2025 Gasto Mercado 4500 Supermercado Compartido Galicia ARS

Ejemplo de fila (USD):
10/02/2025 Ingreso Freelance 500 Pago cliente Mío Galicia USD
